# komand-app

Public assets for [Komand](https://github.com/PauliusOS/komand-app) — the iOS companion app for controlling AI coding agents and model harnesses on your Mac, including Claude Code, Codex, Kimi, Gemini, Cursor and other supported runtimes.

This repository hosts user-facing materials we want to link to from inside the App Store build (paywall, settings, support) and from app store listings. The mobile app and Mac bridge source live elsewhere.
